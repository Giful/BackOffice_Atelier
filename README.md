## GeoQuizz

### Installation

Assurez-vous que le docker fonctionne.

Exécutez les commandes suivantes dans le dossier backoffice_atelier :

``` bash
# Pour installer les dépendances
npm install

# Pour lancer le site sur localhost:8080
npm run dev
```
